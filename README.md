# HYA Svelte WP

## Setup

To install and setup the HYA Gatsby project:

```sh
mkdir hya-wp-svelte.local && cd "$_"
git clone git@github.com:n-d-c/hya-sveltewp.git .
npm install;
npm run dev-- --open;
```

## Serving

To serve the project

```sh
npm run dev-- --open;
```

# Development

```sh
npm run dev-- --open;
```

# Export for Production

```sh
npm run export;
```
